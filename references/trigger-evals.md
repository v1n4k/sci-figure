# Trigger Evals

Use this lightweight checklist when editing `SKILL.md` frontmatter.
The `description` field is the trigger switch: if it is too broad,
the skill over-triggers; if too narrow, it under-triggers.

These are manual eval prompts, not automated tests. Read only the
installed skill metadata (`name` + `description`) and decide whether a
fresh agent should load the skill.

## Expected trigger

| Prompt | Expected | Reason |
| --- | --- | --- |
| "Make a NeurIPS method figure from this manuscript." | trigger | Core use case: one AI/ML manuscript figure. |
| "Design the Figure 1 concept overview for my ML paper." | trigger | Concept / teaser overview is in scope. |
| "Turn this pipeline sketch into an editable drawio figure for the paper." | trigger | Paper figure plus drawio target. |
| "Revise the spacing in the drawio figure generated by sci-figure." | trigger | Existing sci-figure/drawio artifact revision. |
| "Create a main architecture diagram for my ICLR submission." | trigger | Architecture overview for AI/ML manuscript. |
| "I want a teaser figure for the paper, not a method pipeline." | trigger | Teaser / Figure 1 concept figure is explicitly supported. |

## Expected no-trigger

| Prompt | Expected | Reason |
| --- | --- | --- |
| "Draw my architecture in TikZ." | no-trigger | Right figure intent, wrong toolchain. |
| "Can you fix the spacing in this arbitrary PNG?" | no-trigger | Pixel cleanup of non-sci-figure artifact is out of scope. |
| "Make a slide deck diagram for my talk." | no-trigger | Slide decks and non-paper diagrams are out of scope. |
| "What blue should I use for this bar?" | no-trigger | Too small; not a figure-design workflow. |
| "Generate a matplotlib plot for these experiment results." | no-trigger | Raw plotting task, not a drawio manuscript figure. |
| "Make an Inkscape SVG version of this figure." | no-trigger | Different authoring/export toolchain. |

## Conditional / clarify

| Prompt | Expected | Reason |
| --- | --- | --- |
| "Fix the spacing in my existing figure." | clarify | Trigger only if it is a drawio/sci-figure artifact or the user wants a drawio rebuild. |
| "Redraw this PDF figure for my paper." | clarify | Trigger if rebuilding into editable drawio; otherwise use a document/PDF/image workflow. |
| "Make a schematic for my paper." | clarify | Trigger if it is the main AI/ML manuscript figure; skip for a casual sketch. |
| "Improve this architecture diagram." | clarify | Trigger if it targets a paper-ready drawio figure; skip for generic docs/slides. |

When a prompt lands in the conditional band, ask one short scope
question rather than loading every figure-related workflow.
