# sci-figure

Conference-quality main-figure workflow for AI/ML manuscripts.

One invocation produces **one figure** — typically the main method /
pipeline / concept overview the user is currently designing. The
visual story reads in 5 seconds per panel, at the standard expected
by NeurIPS / ICML / ICLR. The deliverable is a `.drawio` source viewed
in the drawio desktop app; auxiliary PNG assets are generated by an
independent matplotlib script and embedded.

## Install

```bash
bash .agents/skills/sci-figure/scripts/bootstrap_env.sh
```

This:
- creates `.venv/` (using `uv` if available, else `python -m venv`),
- installs the Tier 1 plotting stack (`scipy`, `scikit-learn`,
  `seaborn`) plus core (`matplotlib`, `numpy`, `Pillow`, `cairocffi`,
  `CairoSVG`, `joblib`),
- installs `sci-figure-lib` editable (the skill's `lib/`).

## Start a new figure

```bash
bash .agents/skills/sci-figure/scripts/new_figure.sh <name>
cd scripts/<name>
make all
```

Two stubs — `generate_assets.py` and `generate_figure.py` — land in
`scripts/<name>/`. Edit those. The lib stays in the skill.

## Read the workflow
